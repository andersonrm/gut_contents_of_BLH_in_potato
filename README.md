# Molecular Ecology of Host Use and Pathogen Acquisition by an Insect Vector in Potato Crop Fields


This repository contains all data and code used in Wagstaff et al. 2026, Annals of the Entomological Society of America.

## Authors:

[Riley M. Anderson<sup>1</sup>](https://github.com/andersonrm)

Camille Wagstaff<sup>1</sup>

<sup>1</sup>*Dept. of Entomology, Washington State University, Pullman, WA USA*

## Organization:

All data are stored as .csv in the [/data](https://github.com/andersonrm/gut_contents_of_BLH_in_potato/tree/main/data) folder.

All analyses are stored in the [/analyses](https://github.com/andersonrm/gut_contents_of_BLH_in_potato/tree/main/analyses) folder.

Meta data describing all columns can be found in [/data/meta_data](https://github.com/andersonrm/gut_contents_of_BLH_in_potato/tree/main/data/meta_data)
